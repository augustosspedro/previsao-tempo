# previsao-tempo
Aplicação de previsão do tempo que integra as APIs do IBGE (autocomplete de cidades), Nominatim (geocodificação) e Open-Meteo. O projeto tem uma interface moderna em Glassmorphism com animações CSS dinâmicas. Desenvolvido em JS Vanila, utiliza Async/Await para coordenar múltiplas requisições e manipulação de DOM para exibir dados em tempo real.
